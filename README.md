# AoC 2019 IntCode problems: Rust Implementation

## Prerequisites
1. Rust & Cargo

## Build

```sh
cargo build
```

## Run

**Note:** Project has no default binary crate
```sh
cargo run --bin day2 # runs day2
```

## Unit Tests

```sh
cargo test
```
